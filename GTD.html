Graffiti Tower Defense

Custom C++ Engine

Project Information

  • Duration: Febuary 2021 - Present
  • Team Size: 4
  • Platform: Windows PC
  • Engine: Custom C++ Engine
  • Type: Academic Project

Concept

Graffiti Tower Defense is a tower defense style game where the player is trying to protect their wall from the enemy Bloblets. The player will need to buy, upgrade and defeat all of the enemies in each level in order to keep their wall in pristine condition!

My Contribution

  • Built an Input Manager to track per-key input from the player, support for tracking per-frame input or input over multple frames, and mouse position and action information.
  • Using this Input Manager, created a full Raycasting system to allow for direct interation between the player and the 3D game world.
  • Utilized the DirectXTK Audio Engine to make an audio system that can play, pause, loop, edit and fade sound effects and music tracks.
  • Created enemy steering AI with allowed enemies to curve around corners and their nodes, rather than moving to exact node positions.
  • Created an FBX Exporter that can take in an FBX file give a binary text file to import models, textures and animations.
  • Built multiple towers, including a base tower and it's upgrades with unique gameplay mechanics.
  • Created the tower "shake to reload" mechanic.
  • Tracked and organized all enemy, tower, and wave data to properly balance all acpects of the game.
